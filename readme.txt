== Installation ==

1. Upload the entire `featurable-google-reviews` folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the **Plugins** screen in WordPress (**Plugins > Installed Plugins**).

You will find the **Featurable Google Reviews** block in the WordPress block editor (Gutenberg).

== Frequently Asked Questions ==

= What is Featurable Google Reviews? =

Featurable Google Reviews is a free WordPress plugin that allows you to embed Google Reviews on your website.

= How do I get started? =

Simply install the plugin, sign in, and add the block to your website. It's that easy!

= Is Featurable Google Reviews free? =

Yes! We offer a **free lifetime plan with unlimited page views**. You can upgrade to a paid plan for professional features and support.

= Does it work with any WordPress theme? =

**Yes.** Featurable Google Reviews works with any WordPress theme. It's designed to be easy to use and compatible with all themes.

= How does it work? =

The plugin will automatically fetch your latest Google Reviews and display them on your website in a beautiful layout of your choice.

You can customize the design and layout to match your website, as well as filter reviews and more.

= Do I need a Google API key? =

**No**, you do not need a Google API key to use this plugin. We handle all the technical details for you.

= Do I need a Google Place ID? =

**Nope!** Unlike other plugins, you do not need a Google Place ID to use this plugin. We make it easy for you.

Just sign in, choose the business profile you want to use, and you're all set.

= Is this plugin good for SEO? =

Yes! Google Reviews are a powerful form of social proof that can help boost your website's SEO and credibility.

Plus, we automatically add Google structured data to your website, which can help improve your search engine rankings.

= Can I customize the design? =

Yes! You can customize the design and layout of your Google Reviews to match your website.

We are adding more customization options in future updates, including custom CSS and more.
